Hey there!

Thanks for checking out the Rise of Industry Calculator. I wanted a quick and easy tool to figure out how to best meet my town's demands and had some extra time so I put this together. I hope it's useful and if your technical, I'd love your help!

Getting Started

Take the desired input from one of your cities and change the value on the left hand side from 0 to what your city needs. The app should update the production chains as needed.

Current State

So far, this is in a proof-of-concept state, I wanted to get the math worked out, UI figured out, and things of that nature before spending hours copying data from the game. I tried to architect the app in a way that's immediately and easily scalable to support the other production chains and updates as they arrive.

Technical Stuff

I tried to keep this project super easy and "vanilla" so others can very easily help build this out on Github with me. It's simply HTML, CSS, and JS with no server whatsoever.

There are 3 main Javascript files that contain the underlying game logic, "products.js", "buildings.js", and "recipes.js".

I'd love pull requests to add additional data or features, fix bugs, or re-factor things for other contributors!
